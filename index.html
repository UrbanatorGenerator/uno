About Uno

Uno is a personal AI sales assistant built for Powersales Communication. Powered by Claude (Anthropic) and n8n automation, Uno helps with sales strategy, expert advice, research, and business communication.

Web visitors can ask questions and get advice from Uno's expert panel. Some features (calendar, email, contacts) are reserved for admin use.

v4.3.1 · Currently in beta

Privacy & GDPR

Data we collect: Chat messages are processed by Anthropic's Claude API and temporarily stored in session memory (Redis). Sessions expire automatically.

Usage analytics: Anonymised usage statistics (message count, token usage, cost estimates) are logged to a private Google Sheet for operational purposes.

Market data: Financial data sourced from Yahoo Finance and Open Exchange Rates, delayed ~15 minutes.

No personal data is sold or shared with third parties. Questions? urbgav@gmail.com
